# 🧭 AI Tourist Travel Recommendation System

An AI-powered web application that recommends tourist destinations across India based on the user's selected state, tourist category, and best travel season.

---

## 🌐 Live Demo

https://ai-tourist-travel-recommendation-system.onrender.com

---

## 📌 Features

- 🔐 User Login and Registration
- 🏠 Interactive Dashboard
- 🤖 AI-Based Tourist Recommendation
- 🗺️ State-wise Tourist Place Suggestions
- 🖼️ Tourist Images using Pexels API
- 📱 Responsive Design for Mobile and Desktop
- 📞 About and Contact Pages

---

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- Machine Learning
- HTML
- CSS
- Bootstrap 5
- JavaScript
- Pexels API
